# Blinkit Sales & Customer Insights Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard developed to analyze Blinkit's sales performance, customer behavior, product category contribution, and delivery insights.

The dashboard helps stakeholders monitor key business metrics and make data-driven decisions by providing a comprehensive view of sales, profit, customer ratings, and order distribution.

---

## 🎯 Objectives
- Analyze overall sales and profitability.
- Identify top-performing product categories and subcategories.
- Monitor customer ratings and satisfaction.
- Evaluate delivery time performance.
- Understand category-wise sales contribution.

---

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)
- Total Sales: **1M**
- Total Profit: **219.27K**
- Total Orders: **45M**
- Average Customer Rating: **3.08**

### Visualizations
- Sales vs Profit Analysis
- Profit by Sub-Category (Treemap)
- Order Distribution by Delivery Time
- Sales Contribution by Category
- Customer Rating Gauge
- Interactive Filters for:
  - Category
  - Payment Mode
